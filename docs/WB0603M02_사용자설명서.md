# WB0603M02 - 기계별 WBS현황 사용자 설명서

## 📋 프로그램 개요

**프로그램명**: 실행계획 > 기계별 WBS현황  
**프로그램 ID**: WB0603M02  
**목적**: 기계별 WBS(Work Breakdown Structure) 일정을 시각적으로 확인하고 관리할 수 있는 간트 차트 기반 일정 관리 시스템

---

## 🎯 주요 기능

### 1. 검색 조건 설정
- **회사**: 조회할 회사 선택 (필수)
- **수주일자**: 조회 기간 설정 (시작일 ~ 종료일)
- **SalesCode**: 특정 Sales Code로 검색 (검색 아이콘 클릭 시 팝업 검색 가능)
- **고객사**: 고객사명으로 검색 (검색 아이콘 클릭 시 팝업 검색 가능)
- **고객사PJT**: 프로젝트 코드로 필터링

### 2. 간트 차트 보기
- **타임라인 표시**: 평일 기준 일자별 그리드 (주말 제외)
- **월별 구분**: 월 단위로 구분선 표시
- **TODAY 표시**: 현재 날짜 강조 표시
- **부서별 색상**: 각 부서(영업, 설계, 구매, 생산, 전기, 시운전 등)별로 구분된 색상 표시

### 3. 일정 조회 및 표시
- **계획 일정**: 각 단계별 계획 시작일 및 종료일 표시
- **실적 일정**: 실제 진행 상황 표시 (진행률 포함)
- **집계 행**: 수주번호별 전체 일정 요약 표시

---

## 🖱️ 화면 조작 방법

### 검색 및 조회
1. **검색 조건 입력**
   - 회사, 수주일자 등 필수 조건 입력
   - 필요시 SalesCode, 고객사 등 추가 조건 입력

2. **버튼 사용**
   - **초기화**: 검색 조건 초기화
   - **검색**: 조건에 맞는 데이터 조회
   - **도움말**: 프로그램 사용 도움말 표시

### 간트 차트 조작

#### 1. 줌(Zoom) 조절
- **슬라이더 사용**: 화면 상단의 줌 슬라이더를 좌우로 드래그
- **마우스 휠 사용**: `Ctrl` + 마우스 휠로 확대/축소
- **줌 범위**: 2px/day ~ 20px/day

#### 2. 일정 바(Bar) 드래그
- **이동 방법**: 간트 바를 마우스로 클릭한 채 좌우로 드래그
- **효과**: 시작일이 1일 단위로 이동 (기간은 유지)
- **제약**: 평일(업무일) 기준으로 스냅됨

#### 3. 일정 바 리사이즈
- **방법**: 간트 바의 좌측 또는 우측 끝을 드래그
- **효과**: 시작일 또는 종료일 변경으로 기간 조정
- **단위**: 1일 단위로 조정

#### 4. 겹치는 간트 바 선택
- **상황**: 여러 일정이 겹쳐 있을 때
- **방법**: 겹친 영역에 마우스 오버 시 선택 모달 자동 표시
- **선택**: 모달에서 원하는 일정 클릭하여 선택

#### 5. 툴팁 확인
- **방법**: 간트 바에 마우스 오버
- **표시 정보**:
  - 단계명 (예: 영업, 설계, 구매 등)
  - 담당자
  - 상태 (계획, 진행, 완료, 보류)
  - 계획 일정 (시작일 ~ 종료일)
  - 실적 일정 (있는 경우)
  - 진행률 (%)
  - 메모

#### 6. 상세 정보 확인
- **방법**: 간트 바를 더블클릭
- **효과**: WBS 계획 등록 팝업 열림 (상세 정보 확인 및 수정 가능)

## 💾 데이터 저장 및 내보내기

### 1. 변경사항 저장
- **버튼**: `SAVE CHANGES`
- **동작**: 드래그/리사이즈로 변경된 일정을 서버에 저장
- **확인**: 변경된 데이터 JSON 형태로 확인 가능

### 2. 엑셀 내보내기
- **버튼**: `EXPORT EXCEL`
- **형식**: .xlsx 파일
- **내용**: 현재 조회된 WBS 일정 데이터 전체

### 3. 인쇄
- **버튼**: `PRINT`
- **동작**: 현재 화면을 인쇄 미리보기로 표시

## 🎨 화면 구성 요소

### 레전드 (Legend)
화면 상단에 각 부서별 색상 범례 표시:
- **영업** (파란색)
- **설계** (보라색)
- **구매** (녹색)
- **제품입고** (주황색)
- **생산** (빨간색)
- **전기** (청록색)
- **시운전** (갈색)
- **고객검수** (하늘색)
- **출하** (노란색)
- **설치** (파란색)

### 간트 차트 영역
- **좌측 고정 영역**: 기계명, 설비명, SalesCode
- **중앙 타임라인**: 일정 바 표시 영역 (스크롤 가능)
- **우측 고정 영역**: 비고(Remarks)

### 집계 행
- 수주번호별로 전체 일정을 요약하여 표시
- 배경색이 다르게 표시되어 구분 용이

## ⚠️ 주의사항

### 일정 변경 시
1. **업무일 기준**: 주말 및 공휴일은 자동으로 제외됨
2. **1일 단위**: 모든 변경은 1일 단위로 스냅됨
3. **저장 필수**: 변경 후 반드시 `SAVE CHANGES` 버튼 클릭 필요
4. **권한 확인**: 일정 변경 권한이 있는 사용자만 수정 가능

### 툴팁 표시
- 겹치는 간트 바 선택 모달이 열려 있을 때는 툴팁이 표시되지 않음
- 드래그 중에는 툴팁이 숨겨짐

### 성능
- 대량의 데이터 조회 시 로딩 시간이 소요될 수 있음
- 줌 레벨을 낮추면 더 많은 일정을 한 화면에서 확인 가능

## 🔧 단축키 및 팁

### 단축키
- **Ctrl + 마우스 휠**: 줌 인/아웃
- **Enter (검색 필드)**: 검색 실행

### 사용 팁
1. **빠른 조회**: SalesCode나 고객사 필드에서 Enter 키로 즉시 검색
2. **정밀 조정**: 줌을 최대로 올려 일자별 세밀한 조정 가능
3. **전체 보기**: 줌을 낮춰 프로젝트 전체 일정을 한눈에 파악
4. **색상 구분**: 레전드를 참고하여 부서별 진행 상황 빠르게 파악
