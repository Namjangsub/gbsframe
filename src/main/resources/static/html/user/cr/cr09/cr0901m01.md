# 물류진행 예정정보 현황 (CR0901M01) 프로그램 작성 계획서

## 1. 개요
- **프로그램명**: 물류진행 예정정보 현황 (Logistics Progress Scheduled Information Status)
- **ID**: CR0901M01
- **목적**: 
  - 수주 및 WBS 정보를 기반으로 물류가 예정된 품목들을 한눈에 파악.
  - 마감 처리가 완료되었으나 아직 물류 진행이 시작되지 않은 항목들을 필터링하여 업무 효율성 증대.

## 2. 관련 데이터베이스 테이블 및 로직
### 2.1 대상 테이블
- `TB_CR02D04`: 물류예정정보 (Main)
- `TB_CR02D02`: 수주상세정보 (SALES_CD 확인용)
- `TB_WB21M01`: WBS 마스터 (CLOSE_YN, 일정담당자 확인용)
- `TB_CR11M01`: 물류진행마스터 (물류 완료 여부 확인용)
- `TB_CR11D01`: 물류진행상세 (SALES_CD 확인용)
- `TB_CR02M01`: 수주마스터 (영업담당자 확인용)

### 2.2 조회 조건 (SQL 로직)
- **추출 기준**: `TB_CR02D04`에 등록된 정보가 있고, `TRNS_DIV`(운송구분)가 존재하는 자료.
- **필터링 규칙**:
  1. `TB_CR02D02.SALES_CD`가 NULL이 아닌 자료.
  2. `TB_WB22M01.WBSCODE11`이 부여된 자료.
  3. `TB_WB21M01.salesCd`와 매칭되는 항목 중 `CLOSE_YN = 'Y'`(마감 완료)인 자료.
  4. `TB_WB22M01.WBS_PLANE_DT`가 시작일/종료일 범위 내에 있는 자료.
  5. **상태 필터**: 물류진행상세(`TB_CR11D01`) 및 마스터(`TB_CR11M01`) 연계를 통해 '전체', '물류예정', '물류완료' 구분 조회.
  6. **담당자 필터**: 영업담당자 및 일정관리담당자 조회 조건 추가.

### 2.3 SQL 쿼리 설계 (Tibero - CTE 성능 최적화)
```sql
/* 물류진행 예정 정보 조회 쿼리 개요 (CTE 적용) */
WITH WBS_DATA AS (
    SELECT WB22.SALES_CD, WB22.WBS_STA_DT, WB22.WBS_END_DT, WB21.PLAN_MNG_ID
    FROM TB_WB22M01 WB22
    INNER JOIN TB_WB21M01 WB21 ON WB22.SALES_CD = WB21.SALES_CD
    WHERE WB21.CLOSE_YN = 'Y'
      AND WB22.WBSCODE11 IS NOT NULL
)
SELECT D04.*
     , D02.SALES_CD
     , WD.WBS_STA_DT -- 물류예정시작일
     , WD.WBS_END_DT -- 물류예정종료일
     , M01.MNG_ID -- 영업담당자
     , WD.PLAN_MNG_ID -- 일정담당자
FROM TB_CR02D04 D04
INNER JOIN TB_CR02D02 D02 ON D04.CO_CD = D02.CO_CD AND D04.ORDRS_NO = D02.ORDRS_NO
INNER JOIN TB_CR02M01 M01 ON D02.CO_CD = M01.CO_CD AND D02.ORDRS_NO = M01.ORDRS_NO
INNER JOIN WBS_DATA    WD  ON D02.SALES_CD = WD.SALES_CD
WHERE D02.SALES_CD IS NOT NULL
  AND (WD.WBS_STA_DT BETWEEN #{wbsDtFr} AND #{wbsDtTo} OR WD.WBS_END_DT BETWEEN #{wbsDtFr} AND #{wbsDtTo})
  /* 물류 완료 건 Sales CD 기반 제외 로직 포함 */
```


## 3. UI 및 구성 요소
### 3.1 화면 구성 (HTML)
- **위치**: `/static/html/user/cr/cr09/CR0901M01.html`
- **조회 조건 보완**: 
  - WBS 예정일 범위(시작/종료), 고객사, Sales CD, 물류진행상태(전체/예정/완료), 영업담당자.
- **결과 그리드 보완**: 
  - 컬럼: 수주번호, 고객사, Sales CD, **영업담당자**, **일정담당자**, **물류예정시작일**, **물류예정종료일**, **진행상태**, **완료일자**, **물류 상세 정보 전체** 등.
- **상호작용 기능**:
  - **수주내역 조회**: 그리드에서 **Sales CD를 더블클릭**하면 `CR0202P01.html` 팝업을 호출하여 상세 수주 내역을 조회합니다.
  - **물류정보 조회**: `진행상태`가 완료된 행을 더블클릭할 경우 `CR1001P01.html` 팝업을 호출하여 등록된 물류 상세 정보를 조회합니다.
  - **물류상세 등록**: `진행상태`가 완료되지 않은 미완료 건을 더블클릭할 경우 `CR1101P01.html` 팝업을 호출하여 물류진행요청서를 신규 등록합니다.
  - **물류완료/취소 처리**: 그리드에서 복수 행을 선택한 후 상단 버튼을 통해 **'물류완료'** 또는 **'취소처리'**를 수행할 수 있습니다. (진행상태 실시간 업데이트)
  - **엑셀 다운로드**: 조회된 물류 목록 전체를 **엑셀 파일**로 출력하여 로컬에 저장할 수 있습니다.
